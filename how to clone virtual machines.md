[[Initial Windows server and client settings]]
(Questa operazione va fatta nel tuo hypervisor, es. VMware Workstation o VirtualBox, a macchine
spente)
1. Fai clic destro sulla macchina virtuale Tserver -> seleziona Manage -> Clone.
2. Segui la procedura guidata scegliendo Full clone.
3. Nominala Server2019 e concludi la clonazione.
4. Ripeti l'operazione per Tclient: clic destro -> Manage -> Clone -> Full clone -> nominala
Client.
5. Avvia entrambe le macchine clonate.
