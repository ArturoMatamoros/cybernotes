## Struttura 
la nostra cartella, quella in cui ci troviamo attualmente è descritta con: "."
la cartella padre è quindi: ".."

nel caso specifico di [[Linux]] abbiamo una serie di cartelle particolari 
- \bin
- \root
- \dev
	- Qui sono presenti tutti i file (quindi tutto dato che su linux è tutto)
- \etc
- \home
- \lib
- \lost+found
- \usr
- \var
