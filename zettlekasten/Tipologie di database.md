Di [[Database relazionali]] ne esistono svariati:
- [[MySQL]]
	- principalmente usato per siti web e in correlazione con [[PHP]] 
	- non è open source
- MariaDB 
	- è la versione open source di MySQL 
	- usato spesso in server linux 
	- spesso usato in [[Sistema embedded]]  
- [[Microsoft SQL Server]]
	- si usa in windows 
	- permette anche di utilizzare visualizzazione 
	- non open source
- Postgress
- IBM Db2
	- ormai poco utilizzato
- SQLite
	- utilizzato maggiormente all'interno delle app telefoniche
	- possono essere eseguiti all'interno del programma
	- questo è incredibilmente pieno di [[vulnerabilità]]

Esistono anche database non relazionali (noSQL), questi tendono ad essere flessibili e scalabili. Sono utili per strutture con tanti file e con una gerarchia comunque esistente. 
Questi sono molto utilizzati all'interno del [[Cloud Computing]] ed è molto efficiente in quanto distribuito. Non sono per tabelle, sarebbe molto inefficiente. 
I database sono: 
- Key-value store 
- Document store 
- Graph database 

I database cloud sono usati per IOT [[Internet of things]] sono invece: 
- RDS 
- AZURE SQL
- Firestore 

Importante sapere questo dato che durante un [[Penetration test]] abbiamo bisogno di andare a vedere quello che è il database in modo tale da comportarsi in modo diverso
