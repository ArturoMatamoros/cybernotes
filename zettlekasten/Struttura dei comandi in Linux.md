[[Linux]] 
## La struttura 
un comando è composto da 3 parti:
- [[Comandi basilari Linux]]
- Opzioni [[Linux]] (queste iniziano spesso con `-` o `--`)
- Argomenti
Insieme sono hanno la struttura:
``` bash
Comando[opzione][argomento]
```
## Separazione e Concatenazione 
queste ci permettono di creare [[Script Bash]] che rendono infinitamente più rapido effettuare operazioni all'interno del terminale.
- le linee sono separate con `;` 
- possiamo concatenare con `|` (output comando 1 è input del secondo  )
- possiamo salvare l'output in un file utilizzando `>`
