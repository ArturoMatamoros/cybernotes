[[Active directory]]
1. Vai sul Win10client. Accedi (con l'account locale Admin1 creato in precedenza).
2. Assicurati che il server DCPRETEST sia completamente avviato.
3. Sul client, fai clic sul pulsante Start -> Settings -> System -> About.
4. Sulla destra (o in basso), clicca su Advanced system settings.
5. Vai sulla scheda Computer Name e clicca sul pulsante Change....
6. In basso, sotto "Member of", seleziona il pallino Domain: e scrivi ITS.PRI.
7. Clicca OK.
8. Ti chiederà Username e Password. Inserisci l'amministratore del dominio:
○ Username: Administrator
○ Password: (quella del server DCPRETEST)
9. Se tutto è corretto, comparirà il messaggio "Welcome to the ITS.PRI domain". Clicca OK.
10. Ti chiederà di riavviare. Clicca OK, chiudi le finestre e scegli Restart Now.
