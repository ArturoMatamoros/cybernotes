#Arciero
Integrità, sicurezza, riservatezza. Queste sono le tre parole di base della sicurezza. [[Principi della cybersecurity]] Il nostro goal in questo modo è cercare di capire come [[Linux]] interagisce con il hardware. 

Abbiamo questi layer:
- Hardware
	- [[Central processing unit (CPU)]]
	- [[Graphics processing unit (GPU)]]
	- [[Random Access Memory (RAM)]]
	- motherboard 
	- I/O devices 
- [[Kernel]]
-  [[Driver]] 
- Applicazioni
- User

Tieni conto anche del processo di accensione che è composto da: ON, Firmware BIOS/UEFI, ricerca della posizione del GRUB (bootloader che permette il dual boot), kernel.  

Il computer capisce solo binario, ASCII (solo inglese), UTF-8 (tutto), IEEE 754 (reali/naturali).
