## Il linguaggio 
C è un [[Linguaggio di programmazione]] ad alto livello, imperativo, procedurale. C è il linguaggio di più basso livello tra quelli di alto livello. 

Questo linguaggio è molto veloce ed è il linguaggio con cui è scritto il [[Kernel]] in [[Linux]]. Queste sono le ragioni per la quale utilizzare questo linguaggio, principalmente la velocità.
