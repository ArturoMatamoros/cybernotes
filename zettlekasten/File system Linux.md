## Definizione 
il [[File system]] in [[Linux]] è un file system basato sul concetto di [[Inode]] nel [[Allocazione blocchi nel disco]]. In file system abbiamo quello che è il **Virtual file system (VFS)** e quindi possiamo usare le funzioni linux.

Sappiamo [[Struttura File system Linux]] ma dobbiamo sapere cosa è il `mount`che non è altro che un modo di partizioni dei dischi. Questo è utile per capire la struttura modulare.
