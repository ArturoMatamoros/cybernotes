## Struttura grammaticale 
La grammatica di [[Vim]] è incredibilmente semplice: verbo + nome.
Detto questo la grammatica può diventare parecchio dove i nomi sono le motions (i comandi di movimento, che conosco già) ma aggiungo: 
```
$ = fine della linea
{ = fine del paragrafo
```
e i verbi che sono le operazioni e quelle fondamentali sono:
```
y = yank text (copy)
d = delete text 
c = delete text e metti modalità edit 
p = incolla 
```
