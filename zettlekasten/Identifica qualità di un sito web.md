[[IT security]]
Per quanto riguarda siti web sicuri noi abbiamo tanti fattori da tenere conto:
- Tipologia di sito 
- Autore dell'informazione 
- Indicazione delle fonti 
- Confronto di più fonti 
	- Confronto anche con una fonte tradizionale
- L'aspetto del sito web
- HTTPS dove S sta per "Secure" 
	- Presenza del lucchetto
- Certificato di sicurezza
