[[Networking]]
Possiamo considerarlo come una mappatura di un sito web che traduce i nomi di un dominio (google.com) al suo corrispondente [[IP address]] . Parte fondamentali nei [[Protocolli Application]]. 

Un server DNS è un a [[Database]] che contiene la mappatura ed è chiamato Record. Una volta ottenuto l'indirizzo allora il protocollo HTTP può partire.
