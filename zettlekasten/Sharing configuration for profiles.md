[[Roaming profiles and Home directory]]
1. Vai in File Explorer -> Local Disk (C:).
2. Crea una nuova cartella (New -> Folder) e chiamala PROFILIUFFICIO.
3. Fai clic destro sulla cartella -> Properties -> scheda Sharing -> clicca su Advanced
Sharing....
4. Spunta Share this folder.
5. Clicca sul pulsante Permissions, spunta Full Control per Everyone. Clicca Apply e poi OK
fino a chiudere tutte le finestre.
