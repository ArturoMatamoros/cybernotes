[[Processo configurazione network]]
Qui creiamo un [[Domani name system (DNS)]] inserendo un server
- creiamo un server (collegandolo allo switch)
- configuriamo ip ecc. 
- andare su servizi, DNS, e accendere 
- Configurare il nome del sito e il ip corrispondente
- Configurare il dns a quello corrispondente (inserirlo nei computer e nel DHCP)
