## Definizione 
 Git è un **version control systems** che è attualmente lo standard di aversion control. Git è diverso da [[Github]], questi usano git.
 
  Git serve per tenere conto di cambiamenti e ritornare a versioni precedenti dei progetti grazie alle Commit, una serie di cambiamenti che sono salvati e posseggono nome, data e tutte le informazioni necessarie.
