[[file system distribuiti]]
## Definizione 
[[File system]] di rete è un file system che è dato dall'accesso di dati provenienti da macchine diverse. Abbiamo diversi protocolli: 
- network file system (NFS) 
- Samba 
- file transer protocol (obsoleto)

un file system di rete: FS gira sul server. Quindi il server contiene il file system e abbiamo una interazione tra questo è il cliente. Il difetto è che qua è facile che il file system si blocchi per colpa di un traffico di utenti vari.  

Network block device: Fs gira sul client. Il server contiene solo ed esclusivamente dischi. Quindi abbiamo il disco a remoto.
siccome il file system è contenuto nel cliente questo blocco non è accessibile a nessuno se non al cliente stesso.
