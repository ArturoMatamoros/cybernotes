[[Sistema Operativo]]
## Definizione 
Il file system è il metodo di organizzazione di file su un computer. Questi sono organizzati ad albero e ci sono un paio di informazioni da tenere conto:
- esiste sempre una root directory (la cartella iniziale, padre)
- abbiamo una working directory (cartella in cui ci troviamo)
- con / iniziamo un path assoluto 
- con un nome iniziamo un path relativo (che parte dalla working directory)
